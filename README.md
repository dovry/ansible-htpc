ansible-htpc
=========

Fully automated (or as close as I can get it) setup of a home theatre personal computer

A few examples of what that entails

```sh
  # container management
  portainer
  dozzle
  # media streaming
  plex / jellyfin
  # PVR
  sonarr + radarr + lidarr + lazylibrarian
  # Indexers
  jackett
  # download client(s)
  qbittorrent

  and many more
```

Requirements
------------

An Ubuntu or Debian host
