# ansible-htpc

=========

Fully automated (or as close as I can get it) setup of a home theatre personal computer

A few examples of what that entails

## Utilities

- MergerFS + Snapraid
- Samba (NFS)
- Docker

## Containers

### media streaming

- plex / jellyfin

### PVR

- sonarr + radarr + lidarr + lazylibrarian

### Indexers

- jackett

### download client

- qbittorrent

  and many more

## Requirements

------------

An Ubuntu host
